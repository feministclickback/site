Here is where the code for the feministclickback.org site will live.

Tech stack:
- Hugo
- gulp
- Sass
- PostCSS
- Turretcss
- Hosted and deployed on Netlify

## Prerequisites

- Hugo
- Node.js
- npm

## Install

Clone repository

````bash
npm install
````

## Compile assets

````bash
gulp
````

## Development

````bash
hugo server
````

## Production

````bash
hugo
````
